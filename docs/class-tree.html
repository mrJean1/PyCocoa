Class Hierarchy

  • object: The most base type
    • exceptions.BaseException: Common base class for all exceptions
      • exceptions.Exception: Common base class for all non-exit exceptions.
        • exceptions.StandardError: Base class for all standard Python exceptions that do not represent interpreter exiting.
          • exceptions.ValueError: Inappropriate argument value (of correct type).
            • pycocoa.octypes.TypeCodeError: Error in ObjC type encoding.
    • pycocoa.runtime.ObjCBase: Base class for ObjC...
      • pycocoa.runtime.ObjCBoundMethod: Python wrapper for an ObjC class or instance method, an IMP_t.
        • pycocoa.runtime.ObjCBoundClassMethod: Only to distinguish bound class from bound instance methods.
      • pycocoa.runtime.ObjCClass: Python wrapper for an ObjC class.
      • pycocoa.runtime.ObjCInstance: Python wrapper for an ObjC instance.
        • pycocoa.nstypes.CFStr: Python wrapper for the ObjC CFString class, creating retained instances, by default.
          • pycocoa.nstypes.NSStr: Python wrapper for the ObjC NSString class, creating auto-released instances, by default.
            • pycocoa.nstypes.at: Acronym for the Python wrapper of NSStr.
        • pycocoa.nstypes.NSDecimal: Optimized NSDecimalNumber class.
      • pycocoa.runtime.ObjCMethod: Represent an unbound ObjC class or instance method (really an IMP_t).
        • pycocoa.runtime.ObjCClassMethod: Only to distinguish class methods from instance methods.
      • pycocoa.runtime.ObjCSubclass: Python class to create an ObjC sub-class of an existing ObjC (super-)class.
    • _CData: XXX to be provided
      • _ctypes.Structure: Structure base class
        • pycocoa.octypes.c_struct_t: Base type to pretty-print ctypes Structures.
          • pycocoa.octypes.CFRange_t: ObjC struct with fields loc[ation] and len[gth] (both CFIndex_t).
          • pycocoa.octypes.NSPoint_t: ObjC struct with fields x and y (both CGFloat_t).
          • pycocoa.octypes.NSRange_t: ObjC struct with fields loc[ation] and len[gth] (both NSUInteger_t).
          • pycocoa.octypes.NSRect_t: ObjC struct with fields origin and size (NSPoint_t, NSSize_t).
            • pycocoa.octypes.NSRect4_t: ObjC struct, like NSRect_t with different signature and properties.
          • pycocoa.octypes.NSSize_t: ObjC struct with fields width and height (both CGFloat_t).
          • pycocoa.octypes.objc_method_description_t: ObjC struct with fields name and types (SEL_t, c_char_p).
          • pycocoa.octypes.objc_property_attribute_t: ObjC struct with fields name and value (both c_char_p).
          • pycocoa.octypes.objc_super_t: ObjC struct with fields receiver and class (Id_t, Class_t).
      • _ctypes._SimpleCData: XXX to be provided

    • pycocoa.utils._Constants: Only constant, readable attributes.
      • pycocoa.windows.Style: Window style mask constants.
    • pycocoa.bases._Type0: The base Type.
      • pycocoa.dicts.FrozenDict: Python dit Type, wrapping an (immutable) ObjC NSDictionary.
        • pycocoa.dicts.Dict: Python dict Type, wrapping an ObjC NSMutableDictionary.
      • pycocoa.sets.FrozenSet: Python frozenset Type, wrapping an immutable ObjC NSSet.
      • pycocoa.geometry.Point: Python Type, wrapping an ObjC NSPoint_t.

      • pycocoa.geometry.Rect: Python Type, wrapping an ObjC NSRect_t.

      • pycocoa.sets.Set: Python c{Set} Type, wrapping an ObjC NSMutableSet.
      • pycocoa.geometry.Size: Python Type, wrapping an ObjC NSSize_t.
        • pycocoa.geometry.Size2: Python Type, like Size, different signature.
      • pycocoa.strs.Str: Python str Type, wrapping (immutable) ObjC NSStr[ing].
      • pycocoa.tuples.Tuple: Python tuple Type, wrapping an immutable ObjC NSArray.
        • pycocoa.lists.List: Python list Type, wrapping an ObjC NSMutableArray.
      • pycocoa.bases._Type1: Basic Type with app and delegate.
        • pycocoa.menus.Separator: Menu separator Python Type, wrapping ObjC NSMenuItem.separatorItem.
        • pycocoa.bases._Type2: Basic Type with app, delegate, tag and title.
          • pycocoa.apps.App: Application Python Type, wrapping an ObjC NSApplication.
          • pycocoa.menus.Item: Menu item Python Type, wrapping ObjC NSMenuItem.
          • pycocoa.menus.Menu: Menu Python Type, wrapping ObjC NSMenu.
          • pycocoa.menus.MenuBar: Menu bar Python Type, wrapping ObjC NSMenu.
          • pycocoa.panels.OpenPanel: Python Type to select a file, wrapping ObjC NSOpenPanel.
          • pycocoa.panels.SavePanel: Python Type to save a file, wrapping ObjC NSSavePanel.
          • pycocoa.tables.Table: Python Type table of rows and columns, wrapping an ObjC NSTableView.
          • pycocoa.apps.Tile: The dock tile for an App, wrapping an ObjC NSDockTile.
          • pycocoa.windows.Window: Basic, base window Python Type, wrapping ObjC NSWindow.
            • pycocoa.windows.MediaWindow: Media window Python Type, wrapping ObjC NSWindow/NSView.
            • pycocoa.tables.TableWindow: Python Type for a vertically scrollable window, wrapping ObjC NSWindow/NSScrollView.
    • basestring: Type basestring cannot be instantiated; it is the base for str and unicode.
      • str: str(object='') -> string
        • pycocoa.strs.Str: Python str Type, wrapping (immutable) ObjC NSStr[ing].
    • frozenset: frozenset() -> empty frozenset object frozenset(iterable) -> frozenset object
      • pycocoa.sets.FrozenSet: Python frozenset Type, wrapping an immutable ObjC NSSet.
    • set: set() -> new empty set object set(iterable) -> new set object
      • pycocoa.sets.Set: Python c{Set} Type, wrapping an ObjC NSMutableSet.
