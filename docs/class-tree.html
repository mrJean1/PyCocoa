      • exceptions.Exception: Common base class for all non-exit exceptions.
        • exceptions.StandardError: Base class for all standard Python exceptions that do not represent interpreter exiting.
          • exceptions.ValueError: Inappropriate argument value (of correct type).
            • pycocoa.octypes.TypeCodeError: Error in ObjC type encoding.
    • pycocoa.runtime.ObjCBase: Base class for ObjC...
      • pycocoa.runtime.ObjCBoundMethod: Python wrapper for an ObjC class or instance method, an IMP_t.
        • pycocoa.runtime.ObjCBoundClassMethod: Only to distinguish bound class from bound instance methods.
      • pycocoa.runtime.ObjCClass: Python wrapper for an ObjC class.
      • pycocoa.runtime.ObjCInstance: Python wrapper for an ObjC instance.
        • pycocoa.nstypes.CFStr: Python wrapper for the ObjC CFString class, creating retained instances, by default.
          • pycocoa.nstypes.NSStr: Python wrapper for the ObjC NSString class, creating auto-released instances, by default.
            • pycocoa.nstypes.at: Acronym for the NSStr wrapper.
        • pycocoa.nstypes.NSDecimal: Optimized NSDecimalNumber class.
      • pycocoa.runtime.ObjCMethod: This represents an unbound ObjC method (really an IMP_t).
        • pycocoa.runtime.ObjCClassMethod: Only to distinguish class from instance methods.
      • pycocoa.runtime.ObjCSubclass: Use this to create a sub-class of an existing ObjC class.

      • str: str(object='') -> string
        • pycocoa.strs.Str: Python Type equivalent of an (immutable) ObjC NSStr[ing].
