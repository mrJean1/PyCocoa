  • action - Callback, the delegate method to be called (string ending with ':' or _', SEL_t or None).
  • key - The shortcut key, if any (string).
  • alt - Key modifier (bool).
  • cmd - Key modifier (bool).
  • cntl - Key modifier (bool).
  • shift - Key modifier (bool).
